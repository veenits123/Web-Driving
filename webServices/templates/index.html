As per the problem Statement use "chrome" for Chrome and "mozilla" for Firefox browser for rest url filtering

browser and url are the parameters for URL filtering

Cache is cleared when we close and open the browser again

Instructions
http://localhost:8000/start?browser=chrome&url=http://example.com should start Google Chrome and open http://example.com in the same. Similarly for Firefox.
http://localhost:8000/stop?browser=browser should stop the given browser if it is running.
http://localhost:8000/cleanup?browser=browser should clean up the browsing session for the given browser if has been stopped.
